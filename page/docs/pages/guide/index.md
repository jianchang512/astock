# 操作方法：如何根据预测数据买卖股票


### 以 3.20 日生成的预测数据为例，聊聊下个交易日(记为T日)稳健性操作

![](https://pvtr2.pyvideotrans.com/1774107926178_image.png)


> 预测数据生成日期: 3.20日(周五交易日)，预测的是下个交易日 3.23日(21-22周末无交易)

操作时间: T日开盘时

买入策略: Top 5 股票

资金分配: 每只 20%（总共 100%），假设打算买100股，每只买20股

止盈设置: +2%，  假设买入价 10.00 元，止盈价: 10.20 元（+2%）

止损设置: -1.5%，假设买入价: 10.00 元，止损价: 9.85 元（-1.5%）

持有时间: 最多 5 天

买入时间: T日 开盘后（9:30-10:00）
    
    优点: 观察走势后决定
    
    缺点: 可能已经涨了
    
    也可开盘集体竞价时(互有优缺点)


### 操作流程:
1. 买入后立即设置委托
2. 止盈单: 止盈价卖出
3. 止损单: 止损价卖出（二选一触发）

### 持有规则:
- 达到止盈 → 立即卖出 ✅
- 触及止损 → 立即卖出 ❌
- 5天未达目标 → 平仓退出 ⏰
- 大盘暴跌 → 提前止损 ⚠️

### 实际操作时间线:
- T日: 开盘买入
- T日-> T+3日:4个交易日持有观察
  - 每天盘中监控
  - 达到目标就卖
  - 未达目标继续持有
- T+4日: 如果还没卖，全部平仓


> 此为推荐稳健性方案，具体根据喜好和抵抗风险程度变化，例如更激进些只买Top3，止盈+3%,止损-2%，持有7天；或分散操作，Top10全部购买，止盈+1.5%,止损-1%，持有3天。
>
> 具体止盈数据设置，可参考往日回测数据中实际止盈率，Top10对应的各个止盈胜率，70-90%胜率为优秀
> ![](https://pvtr2.pyvideotrans.com/1774103881757_image.png)
>

### 注意预测仅仅是预测，不代表百分百准确

即使回测显示:
- 止盈1%胜率90%
- 止盈2%胜率70%

但仍有失败可能:
- ❌ 10% 的股票可能不涨反跌
- ❌ 30% 的股票可能涨不到2%
- ❌ 市场突发事件导致全部下跌

## 实际操作检查清单


### 交易前 T-1

```
	✅ 下载并查看 2026-03-20_filter_ret.csv
	✅ 选择 Top 5-10 只股票
	✅ 查询股票名称和基本面（避免ST、退市风险）
	✅ 计算每只股票的买入资金
	✅ 设置止盈止损价格
	✅ 检查账户资金是否充足
	✅ 检查是否有足够的可买股数
```
	
### 交易日 T日

```
	9:15-9:25  集合竞价阶段
	├─ 观察选定股票的开盘价
	└─ 判断是否高开过多

	9:30-10:00  开盘后
	├─ 观察走势
	├─ 如果符合预期，下单买入
	└─ 买入后立即设置止盈止损委托

	10:00-15:00  持仓监控
	├─ 观察是否达到止盈
	├─ 观察是否触及止损
	└─ 关注大盘走势

	15:00  收盘
	└─ 记录当天收益情况
```

### 后续交易日 T->T+4
```
每天重复:
├─ 检查是否达到止盈 → 卖出
├─ 检查是否触及止损 → 止损
├─ 观察持股走势
└─ 第5天强制平仓（如果还未卖出）
```


## 常见问题

### Q1: 预测明天涨6%，我能赚到6%吗？

❌ 不一定！

原因:
1. 这是模型的预测，不是保证
2. 实际涨幅可能更高或更低
3. 需要考虑买入价和卖出价

实际操作:
- 设置合理止盈（如2%）
- 不要贪心等待涨6%
- 见好就收

### Q2: 看涨概率100%是不是一定会涨？

❌ 不是！

看涨概率100%的含义:
- 所有模型（如7个）都预测会涨
- 但每个模型的准确率不是100%
- 所以整体也不是100%准确

根据回测数据:
- Top10 止盈1%胜率: 90%
- 说明仍有10%失败概率

### Q3: 我应该买Top几？

根据资金量和风险偏好:

资金 < 5万:
└─ 建议 Top 3-5（精选策略）

资金 5-20万:
└─ 建议 Top 5-10（平衡策略）

资金 > 20万:
└─ 建议 Top 10-20（分散策略）

风险偏好:
- 激进: Top 3
- 稳健: Top 5-10
- 保守: Top 10-20

### Q4: 什么时候买入最好？

最佳时机排序:

1. 开盘后 30 分钟（9:30-10:00）
   ├─ 看清走势
   ├─ 避开集合竞价波动
   └─ 推荐指数: ⭐⭐⭐⭐⭐

2. 集合竞价（9:15-9:25）
   ├─ 以开盘价买入
   ├─ 风险: 可能高开
   └─ 推荐指数: ⭐⭐⭐

3. 尾盘（14:30-15:00）
   ├─ 当天走势明确
   ├─ 风险: 可能已涨太多
   └─ 推荐指数: ⭐⭐

最不推荐: 10:00-14:00（价格波动大）

### Q5: 如何判断预测是否准确？
需要等待后续交易日验证:

3月21日: 买入
3月22日: 观察第1天涨跌
3月23日: 观察第2天涨跌
...

如果:
✅ 大部分股票都在涨 → 预测准确
❌ 大部分股票都在跌 → 预测失败

建议:
- 记录每次操作结果
- 统计自己的胜率
- 调整操作策略
